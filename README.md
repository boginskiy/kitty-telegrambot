# Телеграмм бот «kitty-telegrambot»

### Описание
Бот для релакса. В ответ на ваше сообщение бот присылает картинку с котиком. Каждый раз новую.

### Технологии
* Python 3.7
* python-dotenv 0.19.0
* python-telegram-bot 13.7
